> Systematic German learning platform for Goethe-Zertifikat A1 & A2

[在线访问 / Visit Site](https://leondeutsch0034.github.io/german-learning/)

---

## 这是什么 / About

这是一个面向**中文母语学习者**的德语学习网站,由德语老师 Leon 设计与维护。内容对标**歌德学院 (Goethe-Institut) A1 与 A2** 考试大纲,适合:

- 德语零基础入门者
- 大学德语必修课学生
- 准备 Goethe-Zertifikat A1 / A2 考试的考生
- 想系统补全语法、词汇、听力、口语短板的自学者

A German learning website designed for Chinese-speaking learners, aligned with the **Goethe-Institut A1 and A2 exam syllabi**.

---

## 内容模块 / Modules

- 🔤 **发音训练** — 字母、音素、变音、连读规则
- 📖 **语法系统** — 名词性、变位、格位、从句结构
- 📚 **词汇库** — A1/A2 高频词、主题词汇
- 👂 **听力练习** — 真题级别音频与精听材料
- 🗣️ **口语场景** — 考试 Sprechen 模块训练
- ✍️ **写作模板** — Brief / Email / 表格填写
- 📝 **真题模拟** — 全套模拟题与解析
- 🔍 **Klipp & Klapp 教材专项** — 配套大学德语教材题库

---

## 特色 / Features

- ✅ **十年一线教学方法论沉淀**,非通用 AI 生成内容
- ✅ **中文讲解 + 德语示例**,符合中国学习者认知路径
- ✅ **对标歌德官方考试大纲**,而非泛泛的"德语入门"
- ✅ **单页 HTML 应用**,无需安装,浏览器即开即用
- ✅ **支持手机/平板/电脑**,响应式布局
